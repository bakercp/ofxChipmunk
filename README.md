# A wrapper for Chimpunk2D https://chipmunk-physics.net/ 

## Wrapped features

- [x] Shape
- [x] Body
- [x] Circle
- [x] Rectangle
- [ ] Line
- [x] Convex Polygon
- [x] Concave Polygon
- [x] Spring
- [ ] Gear Joint
- [ ] Pin Join
- [ ] Pivot Joint
- [ ] Rotary Spring
- [ ] Motor

## Platforms

Tested on Linux x64 but should also run on windows and mac.
